# ODS:

04 - Educación de calidad

# Este proyecto analiza la educación en comunidades indígenas de Chiapas, resaltando los factores que limitan el acceso equitativo al aprendizaje. A través de un enfoque reflexivo, se examina cómo el lugar de origen, la pobreza y las barreras lingüísticas influyen en las oportunidades educativas de los estudiantes.
